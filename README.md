## About
iGrave is a project striving to collect manuals from electronics. You can use this product to look for user's manuals that you've lost, and you can also "mourn" for discontinued products.
## Features
- Searching: You can search for products by brand or by model
- Preview: You may preview the machine type by its cover image
- Date of birth: Marks the release date and whether the machine was discontinued
- View PDFs: You can read the user's manuals without downloading
- Crowdsourcing: You can contribute by e-mailing specifications and manual PDFs for machines
